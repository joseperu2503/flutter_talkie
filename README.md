# Talkie

## Compilar android

```bash
flutter build apk --release
```

## Compilar ios

```bash
flutter build ios --release
```

## Cambiar icono de la app

```bash
flutter pub run flutter_launcher_icons
```

## Cambiar splashscreen

```bash
dart run flutter_native_splash:create
```
